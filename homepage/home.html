Welcome to Streamer Wheel Of Names Helper!

Enhance the interaction between streamers and their viewers by providing a dynamic and engaging way though Youtube, Twitch and Discord to manage viewer participation on a wheel of names.

Get started by downloading the latest release and following the installation instructions.

Download Now
Features
  • WheelOfNames Website builtin
  • Twitch Channel Integration
  • YouTube Live Chat Integration
  • Discord Integration
  • Filter User Search Bar
  • Viewer Grid Layout
Chat Commands
  • !wheel
      • Adds the viewer to the wheel
  • !remove
      • Removes the viewer from the wheel
  • !odds
      • Shows the odds of the viewer winning
Technologies Used
  • Vite
  • Vue.js
  • Electron
  • TypeScript
Getting Started

To get started with Streamer Wheel Of Names Helper, download the latest release from the GitHub releases page and follow the installation instructions provided.
